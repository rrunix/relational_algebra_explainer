Database JSON Format

To create a custom database, your JSON file must follow this structure:

{
  "name": "Your Database Name",
  "tables": {
    "TableName1": [
      { "column1": "value1", "column2": 123, ... },
      { "column1": "value2", "column2": 456, ... }
    ],
    "TableName2": [
      { "id": 1, "name": "example", ... }
    ]
  },
  "mermaidSchema": "erDiagram\n...",  // Optional
  "examples": [                        // Optional
    "π[column1](TableName1)",
    "σ[column2>100](TableName2)"
  ]
}

Required fields:

  • name: String - Name of your database
  • tables: Object - Contains your tables as key-value pairs, where each table is an array of row objects

Optional fields:

  • mermaidSchema: String - Custom Mermaid ER diagram (auto-generated if not provided)
  • examples: Array - Example relational algebra queries for this database

Tip: Use "Export Current DB" to see a complete example of the format!
